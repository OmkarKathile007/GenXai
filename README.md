# One Platform Endless AI Possibilities


An all-in-one AI platform designed to accelerate career growth through powerful AI tools for writing, coding, and professional development.

## 🚀 Features

### Core Services
- **Text Summarizer**: Condense content while preserving key information
- **Code Converter**: Translate code between languages with AI-powered accuracy
- **Career Guidance**: Personalized career advice powered by advanced AI
- **Cover Letter Generator**: Instant professional-quality cover letters
- **Improve Text**: Refine and optimize text with AI
- **Email Writing**: Seamless AI email generation tool.

## 🛠️ Tech Stack

- **Frontend**: Next.js, Tailwind CSS, Shadcn UI
- **Authentication**: Clerk
- **Generative ai**: Gemini AI
