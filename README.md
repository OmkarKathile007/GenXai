# GenXai
**One Platform, Endless AI Possibilities**

GenXai is a modern AI-powered career growth platform that brings together multiple advanced tools in one seamless interface. Tailored for students and professionals alike, GenXai accelerates productivity by offering instant text summarization, code conversion, personalized career guidance, and more.


---

## 🔥 Features
- **Text Summarizer**: Condenses content while retaining key information and meaning.
- **Code Converter**: Translates code between languages while preserving logic.
- **AI-Powered Career Guidance**: Personalized career advice and insights.
- **Cover Letter Generator**: Instant generation of professional cover letters.
- **Email Writing Assistant**: Seamless AI email drafting.
- **Text Improver**: Refines and optimizes text for clarity and impact.

## 🎬 Getting Started

### Prerequisites
- Node.js v16+  
- npm or yarn  
- A Clerk account for authentication  
- Google Gemini AI API access  

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/genxai.git
cd genxai

# Install dependencies
npm install 
npm run dev

```
